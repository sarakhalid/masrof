Hyperpay payment Gateway

Fast, Simple and Secure Payments with Hyperpay Payment Gateway

Integrate HyperPay payment Gateway with Odoo and add it as a payment option on your Odoo website for the customers. HyperPay integration allows the website customers to process quick and faster online payments while making purchases from the Odoo website. To ensure faster payments, the customers are not redirected to any other page and the transaction is completed on the Odoo website check out page. The module also maintains the list of transactions in the Odoo backend.

  • Features
    • The module facilitates quick payments for customers on their online orders on Odoo website using Hyperpay payment Gateway
    • Hyperpay Payment shows up as one of the payment methods on the check out page on Odoo website
    • The module assigns easy steps for Hyperpay Payment acquirer integration with Odoo
    • The admin just needs to Merchant ID and authorization from Hyperpay in Odoo
    • On selecting HyperPay, the customer is asked to enter his/her card details to proceed with the payment.
    • The customer needs to fill the required details to complete the transaction
    • After the transaction is complete, the customer is directed back to the Odoo website checkout page
    • The Odoo admin can see the transaction details of the orders in the Odoo backend
    • The order form shows the details of the mode of payment used by the customer to make the payment for his/her order
    • The list of the transaction is also maintained in the HyperPay merchant portal
